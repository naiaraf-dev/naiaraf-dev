## What I Do as a Functional Analyst

In my current role, I work on the **functional side of system development**, acting as a bridge between business and technical teams:

- Collaborating with stakeholders to understand business needs
- Gathering and documenting functional requirements
- Designing system solutions from a functional perspective
- Creating **prototypes and wireframes using Figma** to validate ideas with the business
- Writing functional documentation:
  - Solution proposals
  - Functional specifications for developers
- Performing **functional testing** to ensure systems work as expected

This experience helps me write better code by understanding **why** a system exists, not just **how** to build it.
